# Smart Bookmark App

Smart Bookmark App is a modern web application built with **Next.js 13**, **Supabase**, and **TypeScript** that allows users to save, view, and manage their bookmarks in a secure and real-time environment.  

---

## Features

- Private user authentication using Supabase Auth  
- Add, edit, and delete bookmarks  
- Real-time updates using Supabase Realtime channels  
- Responsive UI with Tailwind CSS  
- Modal-based add bookmark interface  
- Logout functionality  

---

## Tech Stack

- **Frontend:** Next.js 13, React, TypeScript, Tailwind CSS  
- **Backend / Database:** Supabase (PostgreSQL)  
- **Hosting / Deployment:** Vercel
